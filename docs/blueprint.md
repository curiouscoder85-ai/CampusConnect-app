# **App Name**: CampusConnect

## Core Features:

- User Management: Administer user accounts, roles, and permissions within the platform.
- Course Management: Create, edit, and organize course content including videos, assignments, and quizzes.
- Student Dashboard: Provide students with a personalized dashboard to access learning materials, submit assignments, and track progress.
- Feedback Management: Allow teachers to receive and respond to student feedback to improve course quality.
- Course Enrollment: Enable students to browse available courses and enroll in new ones.
- AI Content Suggestion Tool: Leverage generative AI using a Genkit tool to provide students with personalized recommendations for learning materials based on their progress and course content.
- Certificate Generation: Generate and provide certificates of completion to students who finish the course.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke trust, intelligence, and authority.
- Background color: Light gray (#F5F5FA) for a clean and modern look.
- Accent color: Cyan (#00BCD4) for interactive elements and highlights.
- Headline font: 'Space Grotesk' sans-serif for a computerized, techy, scientific feel. Body font: 'PT Sans' sans-serif combining a modern look and a little warmth or personality
- Note: currently only Google Fonts are supported.
- Use modern, line-based icons to represent course categories and actions.
- Implement subtle transitions and animations for user interactions to enhance the overall experience.
